Pair Potentials

The following pair potentials have been coded into pi:

  • Lennard Jones 6-12
    v(r) = 4ε[ (σ/r)12 - (σ/r)6 ]
  • Aziz Potential.
    R. A. Aziz et al., Mol. Phys. 77, 321 (1992).
In addition to these potentials, the Cao-Berne approximation to the hard-sphere pair action has been coded as an empirical pair action.

Coulomb interactions are special because of their 1/r singularities, long range tails, and special symmetry involving the Lens vector. The action for Coulomb problems is discussed in its own section.

Long-range potentials

Long range potentials require periodic images and/or Ewald sumations. Currently these are only implemented for the Coulomb action.

Specifying Interactions in the Input File

To specify the interactions, use the EmpiricalAction tag, 

<!-- Lennard-Jones 6-12 parameters from Seitz and Blancore,
     J. Chem. Thermodynamics, 26 p. 1207, (1996). -->
<EmpiricalAction species1="He" species2="He"
  ngridPoints="1000" rmin="1 pm" rmax="10 nm"
  model="LJ" epsilon="10.22 K" sigma="0.25565 nm"/>
