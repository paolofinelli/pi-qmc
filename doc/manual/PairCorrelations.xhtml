Pair Correlations

General Pair Correlation Function

The pair correlation function (PCF) between two species α and β is

gαβ(r, r') = 〈nα(r)nβ(r')〉/  〈n(r)〉〈n(r')〉.
When the density fluctations are uncorrelated at large distances, such as in a liquid, then g → 1.

Pair Correlations in an Isotropic, Homegeneous System

For an isotropic, homegenous system, the PCF becomes a function of r = |r - r'|, the distance between r and r'.

Static Structure Factor

For a single component with N particles in the periodic cell, the static structure factor is

S(k) = (1/N) 〈nkn-k〉,
where nk = Σj e-ik·rj. Thus S(k) is related to g(r, r') by
S(k) = ∫e-ik·(r-r') [g(r,r')-1] dr.
